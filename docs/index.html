📖 컨셉

Last Banner는 Wesnoth 2D 픽토그램 + CK3/EU4 양식 정치 미학의 변방 영지 운영 로그라이크 전략 게임입니다. 세계는 알아서 돌아가고, 플레이어는 결정 큐에 쌓인 사건만 처리합니다.

시각 톤은 저판타지 political — 마법은 subtle, fireball 같은 화려한 효과는 없습니다. 대신 정치 드라마 + 자원 관리 + 시간 압박이 핵심 긴장감.

🎮 핵심 메카닉

⏯

자동 진행 + 위임

1초 real = 60분 game. 위임하면 3~5초 후 default 결정으로 자동 진행. P 키로 토글.

📜

결정 큐 (11종 사건)

방문객·약탈자·식량 위기·외교·농민 호소·겨울 준비·상인·역병·용병·후계자·건물. 우선순위별 모달.

⚔️

용병 Roster (9-class)

bowman~paladin 9개 클래스 tier 시스템. 충성도/일급/부상/자동 이탈.

👑

왕조 / 후계자 (CK3 양식)

4종 스탯 + loyalty/ambition + 후계자 감사 + 배신 이벤트.

🏗️

빌딩 시스템 (4종)

시장/훈련장/창고/성벽 × max Lv 3. 비용 곡선 + 일간 보너스.

📈

라인 차트

7일 ring buffer. 골드/식량 시계열 + 그림자 + 글리프 도트 + 범례.

🎯 결정 큐 우선순위

LOW MEDIUM HIGH CRITICAL

LOW는 3초 / 그 외는 5초 후 default 결정으로 자동 resolve. CRITICAL은 게임을 즉시 일시정지하여 사용자 결정 필수.

# v4.0.0 신규 autoload (Phase A 4단계) MercenaryData = "res://autoload/mercenary_data.gd" # 9-class tier 카탈로그 UITheme = "res://autoload/ui_theme.gd" # 색상 팔레트 + 스타일 헬퍼 # 코어 autoload GameManager · TimeManager · DecisionQueue · SaveManager GameWorld · EventEngine · AssetRegistry · KoreanFont

게임 내 객체 통계

  • 결정 큐 사건 11종 — VISITOR / BANDIT_RAID / DIPLOMACY / FOOD_SHORTAGE / PEASANT_PETITION / WINTER_PREPARATION / MERCHANT_CARAVAN / PLAGUE / MERCENARY_OFFER / SUCCESSION_AUDIT / HEIR_BETRAYAL / BUILD_CONSTRUCTION
  • 결과 핸들러 27+종 — GameWorld.RESULT_EFFECTS 통합 dict
  • LB_VERIFY 17단계 — 헤드리스 자동 검증 (5~15초)
